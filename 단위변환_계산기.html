CSS 단위 변환 계산기

사용가이드

24px을 rem, em, vw, vh, %로 보고 싶다.

html font-size: 16px
부모 font-size: 16px
시안 사이즈: 1920 × 1080
컨텐츠 영역: 1200px
이라고 하면,

입력:
값(px 기준): 24
root font-size: 16
부모 font-size: 16
화면 너비: 1920
화면 높이: 1080
부모 width: 1200

[변환하기] 버튼 클릭
그러면 결과 영역에:

24px → 몇 rem
24px → 몇 em
24px → 몇 vw
24px → 몇 vh
24px → 부모 1200px 기준 몇 %
